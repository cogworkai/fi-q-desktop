## What is Fi Q?

Fi Q is built around a simple idea:

> **Better financial decisions come from better financial understanding.**

Instead of overwhelming users with spreadsheets, jargon, or fragmented tools, Fi Q focuses on:
- Clarity over complexity  
- Signal over noise  
- Decisions over dashboards

## Core Principles

- **Financial IQ, not financial anxiety**  
  Clear explanations and context-first insights.

- **Actionable by default**  
  Every insight should point to a next step.

- **Minimal, focused, intentional**  
  No clutter. No dark patterns. No gimmicks.
